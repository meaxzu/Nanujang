아나바다 이용 방법

아껴쓰자 (오래 아껴 쓴 물품 등록하기)

사용 방법

내가 오래 아껴 쓴 물건 중 다른 사람에게 나누고 싶은 물건을 등록해요.

  • 물품 등록 시 사용 기간과 물건에 얽힌 이야기(선택) 를 적어주세요.
  • → 다른 사용자들이 물건의 의미를 이해하기 좋아요.
  • 사진은 실제 상태를 보여주는 깨끗한 사진을 올려주세요.

Tip: 아껴 쓴 기간이 길수록 랭킹에 반영돼요 → 다른 사용자들의 관심도가 높아져요.

물품 등록
→
사용 기간 & 이야기 작성
→
깨끗한 사진 업로드

나눠쓰자 (무료 나눔하기)

사용 방법

쓰지 않지만 다른 사람에게 유용할 것 같은 물건을 무료 나눔하고 싶을 때 사용해요.

  • 물품 등록 시 상태 / 사용 가능 여부 / 나눔 조건(직거래, 택배 여부) 를 명확히 적어주세요.
  • 채팅을 통해 필요한 사람과 직접 연락 후 나눔 일정을 조율해요.

Tip: 나눔 완료 후 게시글 상태를 '나눔 완료'로 변경해 주세요.

물품 등록 (상태/조건 명시)
→
채팅으로 나눔 조율
→
나눔 완료 & 상태 변경

바꿔쓰자 (물물교환하기)

사용 방법

내가 가진 물건을 다른 물건과 교환하고 싶을 때 사용해요.

  • 주고 싶은 물건 + 원하는 교환 물건을 명확하게 작성하세요.
  • 교환 조건 / 물건 상태 / 교환 가능 지역도 같이 적어주세요.
  • 채팅으로 교환 조건을 협의하고 → 안전한 장소에서 교환해요.

Tip: 교환할 물건이 여러 개 가능하다면 "교환 가능 목록"을 적어두면 좋아요.

물품 등록 (교환 조건 명시)
→
채팅으로 조건 협의
→
안전한 장소에서 교환

다시쓰자 (재활용 아이디어 공유하기)

사용 방법

재활용하거나 리폼한 물건 / 업사이클링 아이디어를 자유롭게 공유해요.

  • 사진과 함께 설명 / 만들기 팁도 같이 올려주세요.
  • 댓글로 서로 정보 교환 / 칭찬 / 질문 등을 자유롭게 해요.

Tip: 제작 과정 사진이나 전/후 비교 사진을 올리면 반응이 좋아요.

아이디어 & 물건 등록
→
사진 & 설명/팁 포함
→
댓글로 소통

작은 배려가 더 좋은 나눔 문화를 만듭니다.

→ 서로 예의와 배려를 지키며 즐겁게 사용해주세요!
